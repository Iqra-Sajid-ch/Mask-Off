# 🎭 Mask Off - AI Deepfake Detection System

## 📌 Overview

Mask Off is a Final Year Project (FYP) developed as a web-based Deepfake Detection System designed to identify whether uploaded images and videos are authentic or AI-generated. The project utilizes deep learning techniques to assist users in verifying digital media and combating misinformation.

> **Note:** This repository contains the project documentation, frontend interface, and visual assets only. The backend implementation, trained AI models, datasets, and deployment files are intentionally excluded to protect the project's intellectual property.

---

## ✨ Features

- 🔍 Image Deepfake Detection
- 🎥 Video Deepfake Detection
- 📊 Detection Results Dashboard
- 📈 Confidence Score Visualization
- 👤 User Authentication
- 📱 Responsive User Interface
- 🎨 Modern and User-Friendly Design

## 🏗️ System Workflow

1. User uploads an image or video.
2. Media is validated.
3. AI model analyzes the uploaded content.
4. Prediction confidence is calculated.
5. Results are displayed through the dashboard.
